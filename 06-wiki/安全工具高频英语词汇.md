# 安全工具高频英语词汇

> 不用背单词书。每天 5 个，认识就行。

---

## 第一部分：每日学习（菜单 + 协议 + 漏洞词）

### 第一周 — Burp Suite 菜单核心词

#### Day 1：界面基础

| 英文        | 中文  | 在哪看到 |
| --------- | --- | ---- |
| Proxy     | 代理  | 抓包模块 |
| Intercept | 拦截  | 开关按钮 |
| Forward   | 放行  | 按钮   |
| Drop      | 丢弃  | 按钮   |
| Target    | 目标  | 模块名  |

#### Day 2：Repeater + Intruder

| 英文       | 中文   | 在哪看到       |
| -------- | ---- | ---------- |
| Repeater | 重放器  | 核心模块，反复发包用 |
| Request  | 请求   | 发送出去的数据    |
| Response | 响应   | 服务器返回的数据   |
| Intruder | 爆破器  | 批量遍历参数     |
| Payload  | 攻击载荷 | 爆破替换的参数值   |

#### Day 3：操作词

| 英文              | 中文      | 在哪看到            |
| --------------- | ------- | --------------- |
| Decoder         | 解码器     | 编解码模块           |
| Encode / Decode | 编码 / 解码 | 按钮              |
| Filter          | 过滤      | 搜索框/筛选          |
| Scope           | 范围      | 目标范围设置          |
| History         | 历史记录    | HTTP history 标签 |

#### Day 4：设置与插件

| 英文 | 中文 | 在哪看到 |
|------|------|----------|
| Options | 选项 | 设置标签 |
| Settings | 设置 | 菜单 |
| Extender | 扩展模块 | 管理插件的地方 |
| Extensions | 扩展/插件 | 已装插件列表 |
| BApp Store | 插件商店 | 在线装插件 |

#### Day 5：Repeater 发包词

| 英文 | 中文 | 在哪看到 |
|------|------|----------|
| Send / Go | 发送/执行 | Repeater 按钮 |
| Sniper | 狙击手 | Intruder 攻击模式 |
| Attack | 攻击 | Intruder 启动按钮 |
| Position | 位置 | Intruder 标记参数 |
| Follow Redirect | 跟随跳转 | Repeater 选项 |

### 第二周 — HTTP 协议 + 认证

#### Day 6：请求基础

| 英文 | 中文 | 含义 |
|------|------|------|
| Header | 请求头/响应头 | HTTP 消息头部 |
| Body | 消息体 | 请求/响应的正文 |
| Method | 方法 | GET / POST 等 |
| GET | 获取 | 参数在 URL 里 |
| POST | 提交 | 参数在 Body 里 |

#### Day 7：请求头 + 状态码

| 英文 | 中文 | 含义 |
|------|------|------|
| Cookie | Cookie | 登录凭证，存在浏览器里 |
| User-Agent | 用户代理 | 告诉服务器你用什么浏览器 |
| Referer | 来源页 | 从哪个页面跳过来的 |
| Status Code | 状态码 | 200=成功 403=禁止 404=不存在 |
| Redirect | 重定向 | 301/302 自动跳转 |

#### Day 8：数据格式 + 认证

| 英文 | 中文 | 含义 |
|------|------|------|
| Content-Type | 内容类型 | JSON/Form 等格式声明 |
| Authorization | 授权 | 证明你有权限访问 |
| Token | 令牌 | 登录凭证的一种 |
| Session | 会话 | 一次登录连接 |
| Credential | 凭证 | 用户名/密码 |

#### Day 9：漏洞类型（上）

| 英文 | 缩写 | 中文 |
|------|------|------|
| SQL Injection | SQLi | SQL 注入 |
| Cross-Site Scripting | XSS | 跨站脚本攻击 |
| Cross-Site Request Forgery | CSRF | 跨站请求伪造 |
| File Upload | - | 文件上传漏洞 |
| File Inclusion | LFI/RFI | 文件包含漏洞 |

#### Day 10：漏洞类型（下）

| 英文 | 缩写 | 中文 |
|------|------|------|
| Insecure Direct Object Reference | IDOR | 越权访问（改 ID 看别人数据） |
| Remote Code Execution | RCE | 远程代码执行 |
| Server-Side Request Forgery | SSRF | 服务端请求伪造 |
| Privilege Escalation | - | 权限提升 |
| Denial of Service | DoS | 拒绝服务攻击 |

### 第三周 — 攻防实战词

#### Day 11：信息收集

| 英文 | 中文 | 在哪用 |
|------|------|--------|
| Scan / Scanner | 扫描 / 扫描器 | Burp/Nmap |
| Crawl / Spider | 爬取 / 爬虫 | 自动发现页面 |
| Fingerprint | 指纹识别 | 识别网站用了什么技术 |
| Enumeration | 枚举 | 逐个尝试 |
| Reconnaissance | 侦察/信息收集 | 攻击第一步 |

#### Day 12：漏洞挖掘

| 英文 | 中文 |
|------|------|
| Vulnerability | 漏洞 |
| Exploit | 漏洞利用/攻击代码 |
| PoC (Proof of Concept) | 概念验证（证明漏洞存在） |
| Patch | 补丁/修复 |
| Zero-day | 零日漏洞（厂商还没补的） |

#### Day 13：写报告用词

| 英文 | 中文 |
|------|------|
| Severity | 严重程度 |
| Critical / High / Medium / Low | 严重 / 高 / 中 / 低 |
| Impact | 影响（漏洞会造成什么后果） |
| Remediation | 修复建议 |
| Disclosure | 漏洞披露 |

#### Day 14：网络基础

| 英文 | 中文 | 在哪用 |
|------|------|--------|
| Port | 端口 | Nmap/FOFA |
| Service | 服务 | 端口上运行的程序 |
| Banner | 横幅/指纹 | 服务的版本信息 |
| Protocol | 协议 | HTTP/TCP/DNS 等 |
| Asset | 资产 | 目标系统的统称 |

#### Day 15：综合复习

> 前 14 天的词看到英文能反应出中文 → 合格 ✅
> BApp Store 逛一圈看插件名能认出几个 → 检验成果

## 第二部分：BApp Store 常见插件名中英对照

> 逛插件商店时查这个表，不用每个都装，知道是干什么的就行。

### 🔴 必装类（新手三件套 + 爆破）

| 英文名 | 中文解释 | 一句话干什么 |
|--------|----------|-------------|
| **Autorize** | 自动越权检测 | 用低权限 Cookie 重放请求，红了就是漏洞 |
| **Logger++** | 全息流量记录器 | 比原生 History 好用，能搜能导出 |
| **HaE** (Highlighter and Extractor) | 高亮标记与信息提取器 | 自动标出 Token、手机号、密码等敏感信息 |
| **Turbo Intruder** | 涡轮爆破器 | 比自带爆破快几十倍 |

### 🟡 权限 / 认证测试类

| 英文名 | 中文解释 | 干什么 |
|--------|----------|--------|
| Auth Analyzer | 认证授权分析器 | 对比不同用户的返回结果，找授权漏洞 |
| AuthMatrix | 权限矩阵测试 | 用多个账号矩阵式测权限 |
| JWT Editor | JWT 编辑器 | 解码/修改/攻击 JWT 令牌 |
| SAML Raider | SAML 攻击器 | 针对 SAML 单点登录的攻击工具 |
| CSRF Token Tracker | CSRF 令牌追踪 | 自动获取和更新 CSRF Token |

### 🟢 流量分析 / 信息提取类

| 英文名 | 中文解释 | 干什么 |
|--------|----------|--------|
| Flow | 请求链路可视化 | 画出请求之间的关系图 |
| CaA (Collect and Analyze) | 流量收集分析 | 统计接口、参数出现频次 |
| Domain Hunter | 域名猎人 | 从流量里提取子域名、邮箱 |
| JS Miner | JS 挖矿机 | 从 JS 文件里挖隐藏接口和密钥 |
| APIKit | API 文档发现 | 自动发现 Swagger、GraphQL 等 API 文档 |

### 🔵 漏洞扫描增强类

| 英文名 | 中文解释 | 干什么 |
|--------|----------|--------|
| Active Scan++ | 主动扫描增强 | 多检测 RCE、SSRF、XXE 等漏洞 |
| Burp Bounty | 自定义扫描 Payload | 自己加扫描规则 |
| Software Vulnerability Scanner | 软件版本漏洞扫描 | 查网站用的软件有没有已知 CVE |
| Retire.js | JS 库漏洞检测 | 检测页面用的 JS 库有没有漏洞 |
| Backslash Powered Scanner | 反斜杠扫描器 | 用异常行为发现注入点 |

### 🟣 专项漏洞检测类

| 英文名 | 中文解释 | 干什么 |
|--------|----------|--------|
| Param Miner | 隐藏参数挖掘 | 猜出没写在页面上的参数 |
| CORS* | CORS 漏洞检测 | 测跨域配置有没有问题 |
| 403 Bypasser | 403 绕过工具 | 尝试多种方法绕过 403 禁止访问 |
| HTTP Request Smuggler | HTTP 请求走私 | 检测 HTTP 请求走私漏洞 |
| Wsdler | WSDL 解析器 | 解析 Web Service 的 WSDL 文档 |

### 🟠 编码 / 格式化 / 工具箱类

| 英文名 | 中文解释 | 干什么 |
|--------|----------|--------|
| Hackvertor | 编码转换器 | 100+ 种编码互相转，支持嵌套 |
| JSON Beautifier | JSON 美化器 | 把 JSON 格式化成易读的样子 |
| Decompressor | Gzip 解压缩 | 自动解压 gzip 压缩的数据 |
| Knife | 瑞士军刀 | 集成 Hackbar + 域名过滤等小工具 |
| Add Custom Header | 自定义请求头 | 自动给请求加 Header |
| CO2 | 工具集 | SQLMap 命令生成、字典生成等 |

### 🟤 框架漏洞检测类

| 英文名 | 中文解释 | 干什么 |
|--------|----------|--------|
| TsojanScan / BurpShiroPassiveScan | Shiro 漏洞被动扫描 | 检测 Shiro 反序列化漏洞 |
| fastjsonScan | Fastjson 漏洞扫描 | 检测 Fastjson RCE 漏洞 |
| log4j2burpscanner | Log4j2 漏洞扫描 | 检测 Log4j2 核弹级漏洞 |
| J2EEScan | J2EE 漏洞扫描 | 针对 Java Web 应用的专项扫描 |

### ⚪ 联动 / 移动端 / AI 类

| 英文名 | 中文解释 | 干什么 |
|--------|----------|--------|
| sqlmap4burp++ | SQLMap 联动 | 把请求直接发给 SQLMap 跑注入 |
| Brida | Burp 与 Frida 桥梁 | 手机 APP 渗透，过 SSL Pinning |
| autoDecoder | 前端加解密对抗 | 应对前端加密的请求数据 |
| Burp AI Agent | AI 智能代理 | AI 辅助挖漏洞 |
| MCP Server | MCP 服务器 | 让 AI 客户端连接 Burp |

## 学习方法

```
每天 5 个词，不超过 5 分钟
看到英文 → 脑中翻译 → 再看中文确认
第二天先复习前一天的，再学新的
不用背拼写，认识就行
```
