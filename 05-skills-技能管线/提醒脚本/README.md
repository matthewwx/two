# 每日学习提醒小程序

> 到点自动弹出 Windows 桌面通知，提醒学习任务

## 功能

- 每天早上 8:57 弹出提醒
- 可自定义多个时间点
- 最小化到系统托盘，不占资源

## 使用方法

### 1. 运行脚本

```bash
python F:\obsidian\二号大脑\05-skills-技能管线\提醒脚本\daily_reminder.py
```

### 2. 设置开机自启

1. 按 `Win + R`，输入 `shell:startup`，回车
2. 把 `启动提醒.vbs` 的快捷方式拖进去
3. 下次开机自动运行

---

## 自定义

编辑 `daily_reminder.py`，修改 `REMINDERS` 列表：

```python
REMINDERS = [
    ("08:57", "早上好！打开每日学习清单，学5个英语单词，练Burp Suite"),
    ("14:00", "下午了，打开Burp Suite练习抓包吧"),
    ("21:00", "晚上好，刷一篇安全文章，复习今天的单词"),
]
```
