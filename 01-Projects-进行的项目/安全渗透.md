# 安全渗透

> 目标：掌握 Web 渗透测试技能，通过漏洞赏金平台赚取收入

## 学习路线（四阶段，约 6-18 个月）

### 阶段一：筑基期（1-2 个月）

| 模块 | 内容 | 资源 |
|------|------|------|
| 计算机网络 | TCP/IP、HTTP/HTTPS、DNS、Wireshark 抓包 | 《计算机网络：自顶向下方法》 |
| Linux | Kali Linux、文件系统、权限管理、Bash | OverTheWire Bandit 游戏 |
| Windows | 注册表、NTFS、域环境、PowerShell | - |
| 编程 | Python（requests/socket/scapy）、SQL、HTML/JS | 《Python 核心编程》 |
| 法规 | 《网络安全法》、授权原则 | - |

### 阶段二：Web 安全核心（2-4 个月）

**核心工具链：**
- Burp Suite — Web 渗透核心
- Nmap — 端口扫描
- Metasploit — 漏洞利用框架
- SQLMap — SQL 注入自动化
- Gobuster/dirsearch — 目录爆破

**OWASP Top 10 漏洞：**
1. SQL 注入（联合查询、报错注入、布尔盲注、时间盲注）
2. XSS 跨站脚本（反射型/存储型/DOM 型）
3. 文件上传漏洞（MIME 绕过、后缀绕过、图片马）
4. 文件包含（LFI/RFI、伪协议）
5. SSRF 服务端请求伪造
6. CSRF 跨站请求伪造
7. 命令执行/代码执行（RCE）
8. 越权访问（水平/垂直越权、IDOR）
9. XXE 外部实体注入
10. 反序列化漏洞

**靶场练习路径（难度递增）：**
- 入门：DVWA、bWAPP、WebGoat
- 进阶：PortSwigger Web Security Academy、SQLi-Labs
- 综合：Vulnhub（DC 系列）、HackTheBox

### 阶段三：纵深拓展（2-3 个月）

- **内网渗透**：权限提升（Linux/Windows）、域渗透（Kerberoasting、Pass-the-Hash、Golden Ticket、DCSync）
- **横向移动**：psexec、wmiexec
- **工具**：BloodHound、Mimikatz、Impacket
- **代码审计**：PHP/Java 源码审计入门

### 阶段四：综合实战

- **实战平台**：Hack The Box、TryHackMe、VulnHub
- **SRC 平台**：补天、漏洞盒子、阿里先知、腾讯玄武
- **国际平台**：HackerOne、Bugcrowd、Intigriti
- **证书方向**：CISP-PTE → OSCP

## Bug Bounty 赚钱路线

### 新手最易变现的三类漏洞
1. **越权漏洞** — 修改 URL 中的 ID 查看他人数据
2. **密码重置漏洞** — 验证码复用、校验绕过
3. **短信/验证码漏洞** — 爆破、前端泄露

### 收入参考
| 阶段 | 月收入 |
|------|--------|
| 入门（1-3 月） | 0 - 3,000 元 |
| 熟练（半年后） | 3,000 - 15,000 元 |
| 资深（3 年+） | 20,000 - 50,000+ 元 |

### 新手避坑
- ❌ 迷信"一键 Payload"——早被别人交过了
- ❌ 报告写不清楚复现步骤
- ❌ 脱离 Scope 测试——会封号甚至违法
- ❌ 收入稳定前不要辞职全职搞

## 2025-2026 新方向

- 云原生安全（Docker/K8s）
- API 安全（GraphQL、REST）
- 供应链攻击
- AI 安全（对抗样本、模型投毒）
