## 二、国内外 Bug Bounty 平台

### 国内 SRC（推荐入门顺序）

| 平台 | 特点 | 适合阶段 |
|------|------|----------|
| **补天 SRC** | ⭐ 新手首选，审核快，中小厂商多 | 入门 |
| 漏洞盒子 | 项目类型丰富 | 入门 |
| 春秋云测（i春秋） | 定期众测活动，单个严重漏洞可达数万 | 进阶 |
| 阿里先知 | 赏金极高，竞争激烈 | 高手 |
| 腾讯玄武 | 大厂单个漏洞可达数万至数十万 | 高手 |
| 奇安信政企 SRC | 政企项目多，竞争少，单价高 | 进阶 |

### 国际平台（美金结算）

| 平台 | 特点 | 赏金范围 |
|------|------|----------|
| **Intigriti** | ⭐ 新手首选国际平台，审核快（24-48h），不抽成 | €50-€30,000 |
| **Bugcrowd** | 结构化学习路径，反馈详细 | $50-$50,000 |
| **HackerOne** | 全球最大，3000+ 项目，免费 Hacker101 培训 | $50-$100,000+ |
| YesWeHack | 欧洲/亚太，不抽成，有免费 Bug Bounty Dojo 训练 | $50-$30,000 |
| Open Bug Bounty | 无赏金但可积累公开作品集 | 免费练习 |
| Synack | 高赏金私密项目 | $500-$100,000（邀请制） |
| Immunefi | Web3/加密货币专项 | 最高七位数（需专业技能） |

### 新手推荐策略

1. **从 Bugcrowd 或 Intigriti 开始** — 审核最友好，学习资源最多
2. **选新上线、Scope 宽的项目** — 竞争小
3. **优先 SaaS 产品** — IDOR 和认证漏洞多
4. **先读公开报告** — 免费情报
5. **写好报告比挖到漏洞更重要** — 复现步骤清晰、附带 PoC
